本文最后编辑于 前，其中的内容可能需要更新。

( 一 )

    1)了解php开始与结束；

            开始<?php 即系统会自动识别并执行

             结束 ？> 结束标志即结束php代码

       2）了解php的执行块

               有1可知php开始结束格式为<?php ?>

                由此可以发现php执行只与<?php    ?>内的内容有关

                 即发现php执行与<?php？>以外的内容无关

           3）事无绝对在一些情况下php结束标记可以省略而且

            会更好

（二）

            1）由于php是脚本文件所以说php是不需要运行的

            2)因为是脚本文件所以php是可以塞入html的任何地方的

            即一个html文件可以放多个php

            3）php不能因为是脚本文件就随便开个文件夹php必须在

            .php格式中运行的

            4）php可以嵌入其他语言中所以说网页秀不秀看其他的啦

（三）

    1）php中间内容需用分隔符' ;' 来隔离

    2）结束就可以省略 ' ; ' 因为 ?> 含有分隔符

    3）<?php  ?>中间内容不受换行空格的影响

    4） 空格为英文较好中文可能会出现其他错误

（四）

    1）php注释即 /**/ 和 //

    2)/**/为多行注释其两个**中间可以放任意数字字符

    3）// 为单行注释只管一行

· 4）合理运用注释可以方便我们阅读便于理解
（五）
1）变量概念 自已理解可变的量例如衣柜

            啥的可以存放东西  但是编辑语言没有别的只有 

            数据所以不难理解变量即存数据用的

    2）变量的用法 一般语言用变量肯定先说一下再用

    但php不用

    3）php可用作存数据，数字，文本字符串 等数据

    4）存数据用 ‘$+字母下划线数字’ 注意$后第一个需是字母或是下划线开头

    并且不能用空格逗号等将其隔离

    5）定义变量名最好别出现一些常见的关键字防止出现一些不必要

     的错误

    6)函数 echo () 及输出（）中间的内容

    

    7）php变量

    <?php
    $ajb=100;
    $a_j=$ajb;
    echo $a_j;

    $a=aa;
    $$a=199;
    echo $aa
    ?>
    像这种用法也是对的

7)php中删除变量的用法和 起小名的用法

    ?php 
    $a_=199;
    unset ($a_);//既销毁变量名$a_
    echo $a_;//会出现错误因为unset函数已将其销毁
    $asd_=999;
    $ao_=&$asd_;//即给$asd_起个小名
    $ao_=90;
    echo $asd_;//结果会输出90
    ?>
